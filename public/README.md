## Required Image Files (TODO)

You need to add the following image files to this folder:

### 1. Open Graph Image (Social Sharing)
- **File:** `og-image.png`
- **Size:** 1200x630 pixels
- **Format:** PNG
- **Purpose:** Shows when sharing on Twitter, Facebook, LinkedIn, Slack, etc.
- **Design tips:**
  - Include your logo and tagline: "ModelTriage - Right LLM, every time."
  - Use high contrast colors
  - Keep text large and readable
  - Test at https://www.opengraph.xyz/

### 2. Favicon (Browser Tab Icon)
- **File:** `favicon.ico`
- **Size:** 32x32 or multi-size (16x16, 32x32, 48x48)
- **Format:** ICO
- **Purpose:** Shows in browser tabs, bookmarks

### 3. PWA Icons
- **Files:**
  - `icon-192.png` (192x192)
  - `icon-512.png` (512x512)
- **Format:** PNG with transparent background
- **Purpose:** Progressive Web App icons for mobile

### 4. Apple Touch Icon
- **File:** `apple-touch-icon.png`
- **Size:** 180x180 pixels
- **Format:** PNG
- **Purpose:** iOS home screen icon
